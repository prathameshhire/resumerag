# Data Infrastructure Projects

## Log Analytics Pipeline

Built a real-time log analytics pipeline design using Kafka-style event ingestion, stream processing concepts, structured log enrichment, PostgreSQL storage, and query optimization. The design focused on high-throughput ingestion, service-level metadata, and low-latency operational search.

## ETL System

Designed ETL workflows for multi-source data ingestion, transformation, validation, and loading into relational storage. Focused on data quality checks, schema normalization, repeatable jobs, and analytics-ready outputs.

## Observability Metrics

Created operational metrics views for API latency, job failures, and ingestion volume. Used SQL summaries and dashboard-friendly tables to help debug pipeline behavior and identify unreliable data sources.

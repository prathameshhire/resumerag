# Synthetic Resume Sample

## Backend Engineering

Developed REST APIs with FastAPI and PostgreSQL for scheduling, authentication, and document indexing workflows. Worked with SQLAlchemy models, Alembic migrations, and service-layer separation to keep API routes focused.

## Full-Stack Delivery

Built React and TypeScript interfaces for file upload, document listing, semantic search, and generated result review. Used clear loading, empty, and error states to support local developer workflows.

## Data and AI Systems

Created local-first AI features using document chunking, vector retrieval, and Ollama generation. Emphasized privacy by keeping documents, job descriptions, embeddings, and model calls on the local machine.
